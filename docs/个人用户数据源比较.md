## 三、典型使用场景推荐

### 场景 1：个人 A 股日 K 回测
- **免费方案**：Baostock（最简单，数据够用）
- **进阶方案**：JQData 研究版（免费额度足够日常回测）

### 场景 2：分钟级策略回测
- **免费方案**：pytdx（覆盖所有分钟周期）
- **进阶方案**：JQData / Tushare（数据质量更好，分钟级完整）

### 场景 3：实盘数据流
- **免费方案**：pytdx（Level-1 实时行情）+ AkShare（辅助校验）
- **进阶方案**：JQData（稳定 + 合规）

### 场景 4：多品种研究（A 股 + 美股 + 加密货币）
- **免费方案**：AkShare + yfinance（互补）
- **进阶方案**：Tushare（覆盖最广的商业方案）

### 场景 5：基本面 + 量化结合
- **免费方案**：AkShare（财务数据最全的免费源）
- **进阶方案**：Tushare（财务报表、估值、宏观数据最全）

## 四、合规性提醒

### 关键认知

1. **代码开源 ≠ 数据可商用**
   - AkShare（MIT）、yfinance（Apache-2.0）的代码许可证允许商业使用
   - 但这仅意味着你可以用它们的代码，**不意味着你可以用它们获取的数据做商业产品**

2. **免费数据源的本质**
   - 数据最终来源于交易所，交易所通过授权链路分发
   - 免费 API 绕过了授权链路，属于"未被追责"而非"合法"

3. **个人使用的安全边界**
   - 个人研究、学习、回测：**基本无风险**
   - 内部使用、不公开数据：**风险很低**
   - 对外售卖产品、展示行情数据：**需要合规授权**

### 金策智算个人版建议

- 个人版可以内置 pytdx / AkShare 作为默认数据源
- 在文档中声明：**数据源仅供个人研究使用，不得用于商业用途**
- 同时提供 JQData / Tushare 等合规数据源的接入选项
- 如果用户升级到商业版，切换为合规数据源

## 五、接入代码示例

### pytdx

```python
from pytdx.hq import TdxHq_API

api = TdxHq_API()
with api.connect('101.227.73.20', 7709):
    # 获取日 K（category=4 为日线）
    data = api.get_security_bars(4, 1, '600519', 0, 100)  # 上证 贵州茅台
    print(data)
```

### Baostock

```python
import baostock as bs

bs.login()
rs = bs.query_history_k_data_plus(
    "sh.600519",
    "date,code,open,high,low,close,volume",
    start_date='2024-01-01',
    end_date='2025-12-31',
    frequency="d"
)
print(rs.get_data())
bs.logout()
```

### AkShare

```python
import akshare as ak

# 日线数据
df = ak.stock_zh_a_hist(symbol="600519", period="daily",
                        start_date="20240101", end_date="20251231")
print(df)

# 实时行情
df_live = ak.stock_zh_a_spot_em()
print(df_live)
```

### yfinance

```python
import yfinance as yf

# A 股（.SS=上证, .SZ=深证）
maotai = yf.Ticker("600519.SS")
hist = maotai.history(start="2024-01-01", end="2025-12-31")
print(hist)
```

### JQData

```python
from jqdatasdk import *

auth('手机号', '密码')  # 首次注册后获取

# 日 K
df = get_price('600519.XSHG', start_date='2024-01-01',
               end_date='2025-12-31', frequency='daily')
print(df)

# 分钟 K
df = get_price('600519.XSHG', start_date='2024-01-01 09:30:00',
               end_date='2024-01-01 15:00:00', frequency='1m')
print(df)
```

### Tushare

```python
import tushare as ts

pro = ts.pro_api('your_token')  # 注册 tushare.pro 获取

# 日 K
df = pro.daily(ts_code='600519.SH', start_date='20240101', end_date='20251231')
print(df)

# 财务数据
df = pro.fina_indicator(ts_code='600519.SH')
print(df)
```

## 六、总结：个人用户选择路径

```
你是否只需要历史数据回测？
├── 是 → Baostock（零成本，最稳定）
└── 否 → 你需要实时行情吗？
         ├── 不需要 → AkShare（免费，多品种）
         └── 需要 → 你打算商业化吗？
                    ├── 个人用 → pytdx（实时行情 + 分钟 K 最完整）
                    └── 商业用 → JQData / Tushare Pro（合规授权）
```

### 一句话推荐

| 你的情况 | 推荐数据源 |
|---------|-----------|
| 零成本起步，只做历史回测 | **Baostock** |
| 需要免费实时行情 | **pytdx** |
| 多品种快速原型 | **AkShare** |
| 美股/港股为主 | **yfinance** |
| 个人量化投资，愿意花少量钱 | **JQData** |
| 需要全品类数据 | **Tushare Pro** |
| 对外售卖产品 | **JQData / Tushare 企业版** |
